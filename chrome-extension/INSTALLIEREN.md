# ListSync Chrome Extension installieren

## Einmalig installieren (2 Minuten)

1. Chrome öffnen
2. In die Adresszeile tippen: `chrome://extensions`
3. Oben rechts **"Entwicklermodus"** einschalten (kleiner Schalter)
4. Auf **"Entpackte Erweiterung laden"** klicken
5. Diesen Ordner auswählen: `listsync-app/chrome-extension`
6. Fertig – das ListSync-Icon erscheint in der Chrome-Leiste

## Benutzen

1. ListSync starten (`ListSync starten.command` doppelklicken)
2. Listing auswählen → "Crossposten" klicken
3. Vinted oder Kleinanzeigen auswählen
4. Auf "Posten" klicken
5. Chrome öffnet automatisch den richtigen Tab und füllt alles aus
6. Nur noch Bilder manuell hochladen + auf "Veröffentlichen" klicken
